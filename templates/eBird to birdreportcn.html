使用方法：

  1. 务必注意eBird一定要到 "Preferences" 中将 "Species Name Display" 修改为 "Both", 或者 "Common name translated to" 选择 "Chinese(SIM)"。这是因为本项目基于拉丁名或者中文名匹配转换规则。

  2. 从 eBird - 我的记录 下载一个或多个checklist，得到若干名为 xxyyzzww_observations.csv 的文件。

  3. 选中一个或多个csv文件上传，注意不要改变文件名字！

  4. 转换结果为 xxyyzzww_importable.xlsx 或 xxyyzzww_importable_需要手动修复.xlsx，后者需要手动修复一些没能转换的数据。点击链接下载转换结果。

  5. 进入 中国观鸟记录中心，点击“创建观鸟记录 > 定点记”，填写观察点，选择时间，选择“下一步，导入鸟种”，再选择“下一步，上传鸟种数据”，“下一步，预览完成”，即可。

源代码及更多信息请参考 Github 上的项目网站。
